# ros2_template
## Overview
Template repo for ROS 2 with Docker deployment

## Quickstart With Docker
### Installation
```bash
git clone https://github.com/MShields1986/ros2_template.git
# cd ros2_template
# git submodule init
# git submodule update --remote
```

### Execution
```bash
./run.sh
```
